## Deskripsi Proyek
Proyek ini merupakan hasil pengerjaan **Tugas 5 User Interaction** yang bertujuan untuk merancang dan membangun antarmuka pengguna (UI) yang interaktif pada aplikasi berbasis *mobile* menggunakan *framework* Flutter. 

Melalui proyek ini, saya telah mengimplementasikan mekanisme interaksi pengguna (*user interaction*) dan pengelolaan *state* (*state management*) yang terdiri dari:
- **Dynamic ListView**: Menampilkan daftar data (seperti jenis platform dan bahasa pemrograman) yang datanya diambil dari *state* aplikasi dan dapat bertambah secara dinamis.
- **Floating Action Button (FAB)**: Tombol interaktif yang apabila ditekan akan memicu munculnya sebuah *Dialog Box*.
- **Stateful Dialog Form**: Kotak dialog yang berisi form input (judul, platform, bahasa) serta komponen `ToggleButtons` untuk memilih warna latar (*Card color*). Data yang diinputkan melalui form ini akan ditambahkan ke dalam daftar utama secara *real-time*.

Tujuan utama dari tugas ini adalah untuk melatih pemahaman dan penerapan konsep interaktivitas pengguna, penanganan *event* (*onPressed*, *onChange*), serta mekanisme *update state* menggunakan `setState` dan `StatefulBuilder` pada Flutter.
